# Z-MUSIC sample assets

These files are small sample assets used by the mruby-x68k Z-MUSIC examples.

- `simplebgm.ZMD`: normal BGM for `maze_chase.rb` and `zm_game_audio.rb`
- `attackbgm.ZMD`: power-pellet BGM for `maze_chase.rb`
- `eat.ZMD`: dot-eating SE for `maze_chase.rb`
- `charge.ZMD`: power-pellet / ghost-eating SE for `maze_chase.rb`
- `miss.ZMD`: miss jingle for `maze_chase.rb`
- `laser.ZMD`: ZMD SE sample for `zm_game_audio.rb`
- `snare.pcm`: ADPCM sample for `zm_game_audio.rb`

The ZMD files are intended to be played through a resident Z-MUSIC.X driver.
